Cybersecurity Best Practice

Use Strong Password

Never reuse Password across sites and use a password manager.

Avoid Phishing Emails

Don't click on suspicious links or attachments in emails.

Enable 2FA

Always enable Two-Factor Authentication wherever available.

Use Secure Networks

Avoid pubilc Wi-Fi or use a VPN when necessary.

Cyber Hygiene Checklist

  • I use unique Password for every account.
  • I verify email sender before checking links.
  • I regularly update software and system.
  • I backup important data securly.
